# Research Studio

Studio turns verified Project knowledge into typed, editable artifacts rather than opaque generated attachments.

## Artifact types

- Report
- Data table
- Chart
- Timeline
- Comparison matrix
- Mind map
- Evidence map
- Whiteboard
- Slide deck
- Infographic
- Audio overview
- Video overview
- Flashcards
- Quiz
- Code notebook
- Generated file

## Artifact contract

Every artifact has a stable type, structured specification, version history, source and claim dependencies, rendering adapter, export adapters, and validation rules. Artifact components point to Claims and EvidenceSpans; the visual artifact does not invent an independent factual authority.

## Canvas foundations

- Plate for structured editable content.
- Excalidraw for freeform whiteboards.
- React Flow for evidence, dependency, and workflow graphs.
- Vega-Lite for deterministic chart specifications.
- CodeMirror or Monaco for code and structured-data editing.
- Sandpack, JupyterLite, and Pyodide for interactive code artifacts.
- PDF.js, OpenSeadragon, WaveSurfer, Video.js, and HLS.js for evidence viewers.

## Interaction

A user can create an artifact from Chat, a document, selected sources, claims, or a Research Run. The artifact opens in an appropriate canvas with editable structure, evidence inspection, revision history, comments, export, and a path back to the canonical Project knowledge graph.
